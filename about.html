About This Project

This weather dashboard project was created to provide real-time weather information, including temperature, humidity, and wind speed. It also offers a 5-day weather forecast and information on previous weather cautions in the selected city.

Using the OpenWeatherMap API, users can search for any city and get accurate weather data instantly.

The goal of this mini-project is to create a Weather Dashboard that provides real-time weather data, along with additional features such as a 5-day forecast, previous weather cautions, and pages for general information (about, contact). This project involves web technologies like HTML, CSS, and JavaScript to build a fully functional, responsive weather application.

Home Page:
Displays a welcoming interface with a navigation menu to access other parts of the website.
Weather Dashboard (Current Weather):
Displays current weather conditions such as temperature, humidity, wind speed, city name, and weather condition (e.g., cloudy, sunny, etc.). A search functionality allows users to input a city name and fetch real-time weather data using an API.
5-Day Forecast:
Provides the weather forecast for the next 5 days for a selected city. Data includes temperature, humidity, weather conditions, and icons representing the forecasted weather.
Previous Cautions:
Displays past weather cautions (e.g., heatwaves, thunderstorms, air quality warnings) for the selected city. Uses stored data or an API to fetch real-world previous weather warnings.
About Page:
Gives a brief description of the website, its purpose, and how it works.
Contact Page:
Displays contact details or a contact form where users can get in touch with the website owner or team.
